Client X Server

(Voltar)

Afinal, o que é Client? E o que é Server?

Client, nada mais é o computador que solicita e recebe informações de outro computador ( Servidor ).
Exemplos:

  1. Desktop
  2. tablet
  3. notebook
  4. smartphone
    5. etc...

Server, é justamente o computador que armazenaou não, processa e fornece as informações solicitadas ao Client.
Exemplos:

  • Um website que utiliza HTML e CSS
  • Um painel de login
  • Um jogo de perguntas e respostas ( Conferir se as respostas estão corretas. )
    • etc...

Introdução ao HTML

(Voltar)

HTML na prática! Estrutura e principais tags.

A estrutura do HTML é bem simples! Composta por apenas três tags essenciais para seu funcionamento.

  1. <html>
  2. <head>
  3. <body>
Nesta sequência, você tem a estrutura básica para começar seu HTML.

Além dessas, o HTML possui várias outras tags que ajudam na formatação e organização do conteúdo, como:

  • <h1> até <h6>
  • <p>
  • <i>
  • <u>
  • <strong>
  • <ol>
  • <ul>
  • <a>

Glossário

(Voltar)

Esclarecimentos de termos técnicos e explicações.

Ou não: significa que o servidor pode buscar informações em outro sistema, como um banco de dados.
HTML: HyperText Markup Language, ou Linguagem de Marcação de Hipertexto, é a base das estruturas das páginas web.
Tag: Elemento de marcação que define a estrutura e o tipo de conteúdo de uma página HTML.
